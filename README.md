# Arizon E-Commerce

Quality Bearings Online - A modern e-commerce web application built with React.

## Features

- Product catalog with categories
- Product detail pages
- Shopping cart functionality
- Responsive design

## Tech Stack

- React 18.3
- Vite
- Tailwind CSS
- React Router

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/    # Reusable components
├── pages/        # Page components
├── context/      # Cart context
├── hooks/        # Custom hooks
└── utils/        # API utilities
```
